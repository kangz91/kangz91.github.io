■ 기초 금융용어 정리

금융 : 돈의 융통을 줄인 말
금융기관 : 돈을 융통해주는 기관
중앙은행 : 돈을 찍어내는 은행
금융당국 : 금융위원회 + 금융감독원
제1금융권 : 은행
제2금융권 : 은행을 제외환 제도권 금융기관
제3금융건 : 대부업체
여신전문금융회사 : 카드회사, 할부금융회사 등을 가리키는 말


이자 : 돈을 빌려 준 분께 드리는 감사의 마음
금리 : 이자가 차지하는 비율. 예)100원 빌리고 이자 10원을 주었다면 금리는 10%
연리 : 1년동안 내는 이자
단리 : 원금에만 이자가 붙어요
복리 : 이자에도 이자가 붙어요
명목금리 : 통장에 찍혀있는 금리
실질금리 : 물가를 고려했을때 실제로 받는 금리
단기금리 : 1년 안에 갚기로 하고 돈을 빌릴 때의 금리
장기금리 : 1년 이상 쓰기로 하고 돈을 빌릴 때의 금리
장단기금리차 : 장기금리 - 단기금리

 

정책금리 : 정부에서 정책적으로 특별히 관리하는 금리
시장금리 : 정부에서 노터치하는 금리
기준금리 : 한국은행이 일반은행과 돈을 거래할 때의 금리
콜금리 : 은행들 끼리 금전을 빌릴 때의 금리
여신금리 : 대출 해줄 때의 금리
수신금리 : 예금 할 때의 금리
코리보금리 : 은행들끼리 금전을 빌릴 때의 금리
코픽스금리 : 은행이 구한 여러 돈들의 평균 금리

 

채권 : 대힌민국 정부나 각종 회사가 돈을 빌리고 발행하는 차용증
채권금리 :대힌민국 정부나 각종 회사가 차용증을 발행해서 돈을 빌릴 때의 금리
국고채금리 : 대한민국 정부가 돈을 빌릴 때의 금리
회사채금리 : 일반 회사가 돈을 빌릴 때의 금리
양도성 예금증서(CD) : 은행이 돈을 빌리고 발행하는 차용증
CD금리 : 은행이 CD라는 이름의 차용증을 발행해서 돈을 빌릴 때의 금리
신용스프레드 : 회사채금리 - 국고채금리

 

대출금리 : 은행에서 돈을 빌릴 때 적용되는 금리
조달금리 : 은행이 돈을 구해온 평균적인 금리
대출기준금리 : 대출할때 기준이 되는 금리
가산금리 : 사장님 나빠! 사장님은 저번에 돈 잘 안 갚았잖아! 금리 더 받을거야!
고정금리 : 1%의 금리를 주고 빌리기로 했다면 하늘이 무너지기 전에는 계속 1%
변동금리 : 대출받고 내는 이자가 계속 변해요

 

유동성 : 돈. 그리고 돈 비스무리한 무엇
유동성 부족 : 돈이 없어요
통화 : 시중에 유통 되고 있는 돈
통화정책 : 시중에 유통 되고 있는 돈의 양을 조절해서 우리나라 좋은나라를 만드는 정책

 

증권 : 권리를 증명한 종이 쪼가리
유가증권 : 돈으로서의 가치가 있는 종이 쪼가리
주식 : 회사 소유권


재무제표 : 기업이 작성하는 가계부 모음
재무상태표 : 기업의 재산 목록 표
손익계산서 : 1년동안 장사해서 얼마 벌었지?
분식 : 장부 조작
자본 : 돈 버는 돈. 100원으로 사탕 사먹으면 그냥 용돈, 사탕원로 사서 장사하면 자본!
타인자본 : 장사 밑천 중에서 남의 돈
자기자본 : 장사 밑천 중에서 나의 돈
영업이익 : 장사해서 번 돈
당기순이익 : 뗄 것 떼고 순수하게 번 돈

 

연방준비제도 : 미국의 중앙은행
연방준비제도 이사회 : 미국의 중앙은행을 배후(?) 조정 하는 곳
연방공개시장위원회 : 연방기금금리를 결정하는 곳
달러인덱스 : 달러의 전투력을 측정한 것
연방기금금리 : 미국 은행들끼리 급전을 빌릴 때의 금리
미국 국채 금리 : 미국 정부가 돈을 빌릴 때의 금리
리보금리 : 영국 런던에 있는 은행들끼리 돈을 빌릴 때의 금리
