■ 경제지표와 경제

물가, 금리, 환율 등의 주요 경제지표가 가지는 의미와, 이들이 서로 어떤 영향을 주고 받는지 정리 했습니다.

 

경기

경기가 경제에 미치는 영향

경기 → 물가, 경기 → 주가, 기업이익 → 주가, 경제성장률 → 금리

경기가 받는 영향

수출 → 경제, 금리하락 → 소비,투자,경제, 금리상승 → 소비,투자,경제

 

물가

물가가 경제에 미치는 영향

물가 → 주가, 물가 → 금리

물가가 받는 영향

경기 → 물가, 통화 → 물가, 환율상승 → 물가, 환율하락 → 물가

 

금리

기준금리 인상의 의미, 기준금리 인하의 의미, 국채금리 상승의 의미, 국채금리 하락의 의미

금리, 통화량이 경제에 미치는 영향

통화 → 물가, 금리하락 → 소비,투자,경제, 금리상승 → 소비,투자,경제, 기준금리 인하 → 주가, 기준금리 인상 → 주가, 금리 하락 → 주가, 금리 상승 → 주가

금리가 받는 영향

경제성장률 → 금리, 물가 → 금리

 

환율

환율이 경제에 미치는 영향

환율상승 → 수출, 환율하락 → 수출, 환율상승 → 물가, 환율하락 → 물가, 환율 하락 → 주가, 환율 상승 → 주가

환율이 받는 영향

미국 장단기금리차 → 환율, 달러인덱스 → 환율, TED스프레드 → 환율

 

수출

수출이 경제에 미치는 영향

수출 → 경제

수출이 받는 영향

환율상승 → 수출, 환율하락 → 수출, OECD경기선행지수 → 수출

 

세계경제

미국 국채금리 하락의 의미, 미국 국채금리 상승의 의미

해외 경제지표가 한국 경제에 미치는 영향

OECD경기선행지수 → 수출, 달러인덱스 → 환율, 미국 장단기금리차 → 환율, 미국 장단기금리차 → 주가, 미국 금리인상 → 주가, 미국 금리인하 → 주가

 

주가

해외 경제지표로 부터 받는 영향

미국 장단기금리차 → 주가, 미국 금리인상 → 주가, 미국 금리인하 → 주가

국내 경제지표로 부터 받는 영향

경기 → 주가, 물가 → 주가, 기업이익 → 주가, 기준금리 인하 → 주가, 기준금리 인상 → 주가, 금리 하락 → 주가, 금리 상승 → 주가, 환율 하락 → 주가, 환율 상승 → 주가, TED스프레드 → 주가
